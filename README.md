# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
👉 https://manya12381.github.io/Tic-Tac-Toe-Game/

## 📌 Features
- Two-player game (X and O)
- Winner detection
- Reset / New Game button
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
